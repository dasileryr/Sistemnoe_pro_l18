# UserMonitorApp - Мониторинг пользователя

## Быстрый старт

1. Запустите приложение (желательно от имени администратора)
2. Перейдите на вкладку "Настройки"
3. Настройте параметры:
   - Включите статистику и/или модерирование
   - Укажите путь для сохранения отчетов
   - Добавьте запрещенные слова (для модерации)
   - Добавьте запрещенные программы (для модерации)
4. Нажмите "Сохранить настройки"
5. Перейдите на вкладку "Слежение" и нажмите "Запустить мониторинг"
6. Для просмотра результатов перейдите на вкладку "Отчеты"

## Режимы работы

### Статистика
При включенной статистике приложение записывает:
- Все нажатия клавиш в `KeyLog_YYYYMMDD.txt`
- Все запущенные процессы в `ProcessLog_YYYYMMDD.txt`

### Модерирование
При включенном модерировании:
- Обнаруживает запрещенные слова и создает отчеты `ModerationReport_*.txt`
- Блокирует запуск запрещенных программ и создает отчеты

## Важно

- Для работы перехвата клавиатуры требуются права администратора
- Приложение работает незаметно для пользователя (во втором режиме)
- Все отчеты сохраняются в указанной директории
