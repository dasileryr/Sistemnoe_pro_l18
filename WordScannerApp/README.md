# WordScannerApp - Поиск запрещенных слов

## Быстрый старт

### Графический интерфейс
1. Запустите приложение
2. Введите запрещенные слова (по одному на строку) или загрузите из файла
3. Укажите выходную директорию
4. Выберите расширения файлов для поиска
5. Нажмите "Старт"

### Командная строка
```bash
WordScannerApp.exe --no-gui --words "words.txt" --output "C:\Output" --extensions ".txt,.doc"
```

## Результаты работы

После завершения сканирования в выходной директории будут созданы:
- `original_<имя_файла>` - копии оригинальных файлов с запрещенными словами
- `modified_<имя_файла>` - файлы с заменой запрещенных слов на `*******`
- `Report_YYYYMMDD_HHmmss.txt` - подробный отчет о сканировании

## Особенности

- Многопоточная обработка файлов (до 4 потоков одновременно)
- Возможность приостановить/возобновить/остановить сканирование
- Прогресс-бары для отслеживания процесса
- Поддержка различных кодировок файлов
- Игнорирование системных и скрытых директорий
